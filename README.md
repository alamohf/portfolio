# 🚀 Meu Portfólio Pessoal

Este é o resultado do desafio prático de desenvolvimento web, onde construí um site de portfólio/currículo do zero utilizando apenas tecnologias fundamentais da web.

## 📋 Sobre o Projeto

O objetivo deste projeto foi praticar a autonomia no desenvolvimento, utilizando a documentação oficial (W3Schools) para estruturar informações pessoais, habilidades e projetos de forma semântica e responsiva.

### 🛠 Tecnologias Utilizadas

* **HTML5**: Estruturação semântica das seções.
* **CSS3**: Estilização moderna utilizando o conceito de **vCard** (cartão flutuante) e **Flexbox** para alinhamento.
* **Git & GitHub**: Controle de versão e hospedagem.

## 💻 Seções do Site

- **Cabeçalho:** Nome e cargo atual (Estudante de ADS).
- **Sobre Mim:** Breve resumo profissional e objetivos.
- **Habilidades:** Listagem de tecnologias dominadas.
- **Portfólio:** Seção de projetos (Ex: EMI - online, Barber Connect) com links diretos para repositórios e demos.
- **Contato:** Canais diretos via E-mail e GitHub.

## 📝 Aprendizados

Durante este desafio, foquei em:
1.  Centralização de elementos com `Flexbox`.
2.  Criação de imagens circulares com `border-radius`.
3.  Organização de layout sem quebras utilizando `flex-wrap`.
4.  Boas práticas de commit e documentação.
